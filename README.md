Driver Monitoring and Safety System

Overview
This project presents a driver monitoring system for autonomous and semi-autonomous vehicles. It detects driver distraction, behavior, attention, and posture using deep learning and computer vision models. The study compares multiple models to identify the most suitable approach for real-time driver safety monitoring.

Features
- Driver Distraction Monitoring
- Driver Behavior Monitoring
- Driver Attention Detection
- Adaptive Autopilot Safety Monitoring
- Model Performance Comparison

Models Used
- Vision Transformer (ViT)
- Swin Transformer
- YOLOv8 Nano
- MediaPipe Face Mesh
- OpenPose

Tools & Technologies
- Python
- Google Colab
- PyTorch
- OpenCV
- MediaPipe
- YOLOv8
- IBM SPSS
- Kaggle Dataset

Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score

Results
- Swin Transformer outperformed Vision Transformer for driver distraction monitoring.
- YOLOv8 Nano achieved the best overall performance for driver behavior, attention, and safety monitoring.
